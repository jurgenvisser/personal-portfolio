# Jurgen Visser Portfolio

One-page personal portfolio website for Jurgen Visser.

Built with React, Vite, and Tailwind CSS. Ready for deployment on Cloudflare Pages.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

The static build is written to `dist`.

## Cloudflare Pages

- Build command: `npm run build`
- Build output directory: `dist`

Version: `PPJV.0.0.1P`
